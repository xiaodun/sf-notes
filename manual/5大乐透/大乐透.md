# 大乐透

大乐透号码预测工具，支持自动生成号码、查询开奖结果、自动匹配计算中奖情况。

## 创建预测

在预测列表页面，点击"新增预测"按钮，输入预测名称（自动填充当前日期），创建新的预测记录。

## 生成投注号码

进入预测详情页，可以生成投注号码。

### 设置固定号码（可选）

如果你有幸运数字，可以设置固定号码：

1. 在"固定号码"设置中，输入你想要固定的号码
2. 可以设置多个固定号码组合

### 生成随机号码

1. 点击"生成号码"按钮
2. 选择要生成的数量（可以一次生成多组）
3. 系统会自动生成符合大乐透规则的号码：
   - 前区：从 1-35 中选择 5 个不重复的数字
   - 后区：从 1-12 中选择 2 个不重复的数字

### 管理号码列表

- 查看所有已生成的号码组合
- 删除不需要的号码
- 继续添加更多号码

## 查询开奖结果

### 设置开奖日期

1. 在预测详情页，找到"开奖日期"选择框
2. 点击下拉框，会显示最近一个月的开奖日期（大乐透每周一、三、六开奖）
3. 选择你要查询的开奖日期

### 获取开奖号码

1. 点击"获取中奖号码"按钮
2. 系统会自动从体彩网官方 API 获取真实开奖数据
3. 开奖号码会固定显示在页面顶部：
   - 前区 5 个号码高亮显示
   - 后区 2 个号码高亮显示

## 自动匹配计算

获取到开奖号码后，系统会**自动**将你投注的所有号码与开奖号码进行匹配计算：

1. **自动匹配**：系统自动计算每注号码的前区和后区匹配数量
2. **自动判断中奖等级**：根据大乐透规则自动判断中奖等级（一等奖到九等奖）
3. **自动显示奖金**：自动显示对应中奖等级的奖金金额

### 中奖等级规则

- 一等奖：前区 5 个 + 后区 2 个
- 二等奖：前区 5 个 + 后区 1 个
- 三等奖：前区 5 个 + 后区 0 个
- 四等奖：前区 4 个 + 后区 2 个
- 五等奖：前区 4 个 + 后区 1 个
- 六等奖：前区 3 个 + 后区 2 个
- 七等奖：前区 4 个 + 后区 0 个
- 八等奖：前区 3 个 + 后区 1 个，或前区 2 个 + 后区 2 个
- 九等奖：前区 3 个 + 后区 0 个，或前区 1 个 + 后区 2 个，或前区 2 个 + 后区 1 个，或前区 0 个 + 后区 2 个

### 结果展示

- 每注号码的匹配情况一目了然
- **中奖号码用不同颜色高亮显示**，未中奖号码正常显示
- **中奖等级用彩色标签展示**：
  - 一等奖：红色标签
  - 二等奖：橙色标签
  - 三等奖：金色标签
  - 其他等级用不同颜色区分
- 奖金金额清晰标注

## 生成投注二维码

投注前，可以生成包含所有投注信息的二维码：

1. 点击"生成二维码"按钮
2. 弹出二维码弹窗，显示包含所有投注信息的二维码
3. 二维码内容包含：
   - 固定号码（如果有，可选择是否包含）
   - 随机号码
   - 投注金额
4. 方便在手机上扫描查看，直接用于投注

## 核心优势

- ✅ **真实数据**：直接从体彩网官方 API 获取开奖号码，数据真实可靠
- ✅ **自动匹配**：无需手动对号码，系统自动匹配并计算中奖等级
- ✅ **智能计算**：根据大乐透规则自动判断中奖等级和奖金
- ✅ **批量处理**：支持批量生成号码，批量匹配计算
- ✅ **零错误率**：系统自动计算，不会出错，不会漏掉中奖号码

**彩票有风险，投注需谨慎。这个工具只是帮助选号和查询中奖情况，不保证中奖**
