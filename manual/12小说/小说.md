# 小说

小说阅读和管理工具，支持添加本地小说、自动格式化文本、记录阅读进度、自动加载下一章等功能，提供流畅的阅读体验。

# 功能特性

- **小说管理**：添加、删除小说，支持本地文件夹路径
- **智能格式化**：按照中文阅读习惯自动格式化文本内容
- **阅读进度**：自动记录和恢复阅读进度
- **章节导航**：快速跳转到任意章节
- **自动加载**：阅读到章节末尾自动加载下一章
# 添加小说

点击页面底部"添加"按钮，在弹出的模态框中：

- **小说名称**：输入小说的名称
- **路径**：点击"选择路径"按钮选择本地小说文件夹
  - 文件夹应包含按章节命名的文本文件（如 `1.txt`, `2.txt`, `3.txt`...）
  - 章节文件按文件名排序，第一个文件为第一章
- 添加成功后会显示在小说列表中

# 小说列表

小说列表页面显示所有已添加的小说：

- **小说名称**：点击任意行可以进入阅读页面
- **操作按钮**：
  - **删除**：删除小说（删除操作不需要二次确认）
  - **修改**：可以修改小说名称和路径

# 阅读页面

## 章节选择器

- **位置**：固定在页面右上角（移动端适配为右上角）
- **显示逻辑**：
  - 初始显示，3秒后自动隐藏
  - 向上滚动时显示
  - 向下滚动时立即隐藏
- **章节搜索**：可以直接输入章节号（如输入"1"匹配"第1章"）
- **章节信息**：选择器显示当前章节/总章节数（如"27 / 300"）

## 阅读体验

### 自动格式化

文本内容按照中文阅读习惯自动格式化：

- **段落分隔**：段落之间自动添加空行
- **引号处理**：
  - 完整的引号对（`""`、`''`、中文引号）保持在同一行
  - 引号内的内容压缩为单行
  - 多个引号对话正确分割为独立行
- **括号处理**：
  - 单个`【】`与文本一起显示
  - 多个连续的`【】`每个单独成行
- **句子分割**：按标点符号（`。！？；：`）自动分割句子，每个句子独立一行
- **省略号处理**：省略号不触发句子分割，保持连贯性


### 阅读进度

- 自动保存当前阅读的章节
- 下次打开小说时，自动跳转到上次阅读的章节
- 如果没有阅读记录，默认从第一章开始
