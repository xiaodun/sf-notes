# 小说

小说阅读和管理工具，支持添加本地小说、自动格式化文本、记录阅读进度、自动加载下一章等功能，提供流畅的阅读体验。

# 功能特性

- **小说管理**：添加、删除小说，支持本地文件夹路径
- **阅读进度**：自动记录和恢复阅读进度
- **章节导航**：快速跳转到任意章节
- **自动加载**：阅读到章节末尾自动加载下一章
# 添加小说

点击页面底部"添加"按钮，在弹出的模态框中：

- **小说名称**：输入小说的名称
- **路径**：点击"选择路径"按钮选择本地小说文件夹
  - 文件夹应包含按章节命名的文本文件（如 `1.txt`, `2.txt`, `3.txt`...）
  - 章节文件按文件名排序，第一个文件为第一章
  - **支持文件夹**：支持直接读取包含文本文件的文件夹。
- 添加成功后会显示在小说列表中

# 小说列表

小说列表页面显示所有已添加的小说：

- **小说名称**：点击任意行可以进入阅读页面
- **操作按钮**：
  - **删除**：删除小说（删除操作不需要二次确认）
  - **修改**：可以修改小说名称和路径

# 阅读页面

## 章节选择器

- **位置**：固定在页面右上角（移动端适配为右上角）
- **显示逻辑**：
  - 初始显示，3秒后自动隐藏
  - 向上滚动时显示
  - 向下滚动时立即隐藏
- **章节搜索**：可以直接输入章节号（如输入"1"匹配"第1章"）
- **章节信息**：选择器显示当前章节/总章节数（如"27 / 300"）

## 阅读体验

### 阅读进度

- 自动保存当前阅读的章节
- 下次打开小说时，自动跳转到上次阅读的章节
- 如果没有阅读记录，默认从第一章开始

# 页面导航

- **返回主页**：在 PC 端页面底部增加了“返回”按钮，点击可快速返回主页。为了保持界面简洁，该按钮在移动端会自动隐藏。
