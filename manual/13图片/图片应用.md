# 图片应用使用文档

## 1. 功能概述

图片应用是一个用于处理图片的工具，支持以下功能：

- 图片上传
- 图片裁剪
- 图片压缩
- 图片下载
- 图片覆盖

## 2. 界面说明

### 2.1 上传区域

- 左侧显示一个加号按钮，点击可以上传多张图片
- 上传后，图片会在左侧显示预览，点击可以切换要处理的图片

### 2.2 编辑区域

- 右侧显示当前选中图片的编辑界面
- 包含裁剪功能，可以通过拖动裁剪框来调整裁剪区域
- 包含压缩功能，点击"压缩图片"按钮可以压缩当前图片
- 包含图片名称编辑功能
- 包含下载和覆盖按钮

## 3. 使用方法

### 3.1 上传图片

1. 点击左侧的加号按钮
2. 在弹出的文件选择对话框中选择要上传的图片
3. 图片会自动上传并显示在左侧的预览列表中

### 3.2 选择图片

- 点击左侧预览列表中的图片，右侧会显示该图片的编辑界面

### 3.3 裁剪图片

1. 在右侧编辑界面中，拖动裁剪框的边缘或角落来调整裁剪区域
2. 点击"确认裁剪"按钮，完成裁剪操作
3. 裁剪后的图片会显示在编辑界面中

### 3.4 压缩图片

1. 点击"压缩图片"按钮
2. 系统会使用默认参数（质量80，不缩放）对图片进行压缩
3. 压缩后的图片会显示在编辑界面中
4. 界面会显示压缩前后的图片尺寸和大小信息

### 3.5 下载图片

1. 点击"下载"按钮
2. 图片会以当前名称下载到本地

### 3.6 覆盖图片

1. 在"图片名称"输入框中输入新的名称
2. 点击"覆盖"按钮
3. 系统会将当前编辑的图片覆盖原图片
