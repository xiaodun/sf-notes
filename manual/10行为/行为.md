# 行为

行为记录和管理工具，支持创建行为、添加打卡记录、标签管理等功能，帮助您追踪和管理日常行为习惯。

# 功能特性

- **行为管理**：创建、编辑、删除行为，支持加密/解密切换
- **加密支持**：支持对行为进行加密，保护隐私数据，包括行为名称、创建时间、更新时间、打卡记录、标签等
- **打卡记录**：记录行为发生的时间、描述和关联标签，支持修改和删除
- **标签系统**：支持全局标签和行为专属标签，支持数值型和布尔型标签
- **时间轴展示**：可视化时间轴，显示打卡记录之间的时间间隔


# 关于加密
 
  - 密码存在内存中，不进行记录，涉及加密时，没有则创建。
  - 有加密记录则第一次需要输入密码，正确与否以显示为准，应用内跳转只需要输入一次，刷新页面失效。

# 创建行为

点击页面底部"添加"按钮，在弹出的模态框中：

- **活动名称**：输入行为的名称
- **加密选项为是**：
  - 如果当前没有其他加密行为，会生成新密码并显示在模态框中
  - 如果已经有加密行为，但是没验证则会让输入密码

- 创建成功后会自动跳转到行为详情页

# 行为列表

行为列表页面显示所有已创建的行为：

- **行为名称**：点击可进入详情页
- **加密标识**：如果行为已加密，会在名称后显示"加密"标签
- **创建时间**：显示行为的创建时
- **操作按钮**：
  - **修改**：可以修改行为名称和加密状态
  - **删除**：删除行为及其所有相关数据
- **展示加密笔记**：如果存在加密行为，底部会显示"展示加密笔记"按钮
  - 点击后需要输入密码验证
  - 验证成功后，加密行为会显示在列表中（
  - 再次点击可以隐藏加密行为

# 修改行为

点击行为列表中的"修改"按钮，可以修改行为：

- **活动名称**：可以修改行为名称
- **是否加密**：可以切换加密状态


## 打卡记录

详情页以时间轴形式展示所有打卡记录：

- **时间轴**：左侧竖线连接所有打卡记录
- **时间节点**：每个打卡记录前有一个圆形节点，位于两个记录之间的中间位置
- **时间差显示**：节点上方显示与上一条记录的时间间隔
  - 自动选择合适的时间单位（分钟、小时、天、月、年）
  - 如果时间差大于等于1天且还有小时数，会同时显示天数和小时数（如"9天5小时"）
  - 时间差计算基于打卡记录的日期时间
- **打卡信息**：每条记录显示日期时间、关联标签、描述内容
  - 标签会换行显示，格式为"标签名称: 标签值"
  - 加密行为的标签名称和值会被解密后显示
- **操作按钮**：可以修改或删除打卡记录

## 添加打卡记录

点击底部"打卡"按钮，在弹出的模态框中：

- **日期时间**：选择打卡的日期和时间（精确到小时）
- **标签选择**：
  - 可以多选全局标签和行为专属标签
  - 全局标签会显示 `[全局]` 前缀
  - 加密行为的标签名称会被解密后显示
- **标签值设置**：
  - **数值型标签**：输入数值（最小值为1，不能为负数）
  - **布尔型标签**：开关切换（是/否）
- **描述**：可选的文字描述
- 加密行为的描述和标签值会自动加密存储

## 管理标签

### 全局/行为专属标签

在行为列表页面点击底部"标签"按钮，可以管理全局标签：

- 全局标签可以在所有行为中使用
- 支持添加、编辑、删除标签
- 标签类型：数值型、布尔型
- 添加标签时：
  - **标签名称**：输入标签名称
  - **标签类型**：选择"数值"或"是否"
  - **默认值**：数值型标签可以设置默认值（最小值为1）
